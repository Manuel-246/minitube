# MiniTube Pro 🎥

MiniTube Pro is a modern YouTube-inspired video streaming application built using React and Vite. It allows users to search videos, browse content, and enjoy a clean, responsive interface.

## Features

- 🔍 Video Search
- 🎬 Video Player
- 📱 Responsive Design
- ⚡ Fast Performance with Vite
- 🎨 Modern User Interface
- 🌙 Clean and User-Friendly Layout

## Technologies Used

- React.js
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3
- YouTube Data API v3

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/minitube-pro.git
```

2. Open the project folder

```bash
cd minitube-pro
```

3. Install dependencies

```bash
npm install
```

4. Add your YouTube API Key

Create a `.env` file in the root directory and add:

```env
VITE_YOUTUBE_API_KEY=YOUR_API_KEY_HERE
```

5. Start the development server

```bash
npm run dev
```

6. Open in browser

```text
http://localhost:5173
```

## Build for Production

```bash
npm run build
```
